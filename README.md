# 🪪 Dev ID Card

Projeto de cartão interativo de desenvolvedor com foco em **identidade visual, UI moderna e apresentação profissional**.

---

## 🚀 Sobre o projeto

O **Dev ID Card** foi desenvolvido com o objetivo de criar uma apresentação visual impactante, simulando um "cartão de identificação" digital para desenvolvedores.

A ideia principal é transmitir, de forma simples e direta:

* Quem eu sou
* Minha área de atuação
* Tecnologias que utilizo
* Formas de contato

Tudo isso com uma estética moderna inspirada em interfaces tecnológicas.

---

## 🧠 Motivação

Este projeto foi criado para praticar:

* Estruturação com HTML
* Estilização avançada com CSS
* Organização de layout
* Criação de identidade visual

Além disso, foi pensado como um projeto de portfólio para demonstrar habilidades em front-end.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎯 Funcionalidades

* Layout moderno e responsivo
* Identidade visual com tema dark + neon
* Seção com tecnologias utilizadas
* Links para redes sociais
* Estrutura preparada para futuras animações (ex: flip card)
