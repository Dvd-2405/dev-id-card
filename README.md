## 🧩 Sobre o Projeto

O Dev ID Card é um cartão de identificação digital para desenvolvedores. Com um clique, o cartão vira revelando informações de contato no verso — tudo construído com CSS 3D puro e JavaScript vanilla.

A proposta foi criar uma apresentação profissional com identidade visual forte: tema dark, acentos neon verde e tipografia terminal.

---

## 🚀 Funcionalidades

- 🔄 Animação 3D flip ao clicar no cartão (CSS `rotateY` + `perspective`)
- 🎨 Estética neon terminal com cor primária `#00FF87`
- 🔗 Links reais para LinkedIn, GitHub e e-mail
- 📱 Layout responsivo para mobile e desktop
- 🔡 Tipografia com JetBrains Mono e Space Grotesk

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3 (animação 3D, tema dark/neon, Flexbox)
- JavaScript Vanilla (`classList.toggle` para controle do flip)
- Fontes: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) + [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)
